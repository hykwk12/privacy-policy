개인정보 처리방침

본 페이지는 “고전번역 저장기” ChatGPT 플러그인의 개인정보 처리 방침을 안내합니다.

1. 개인정보 수집 항목

본 플러그인은 Google Sheets API 및 OpenAI API를 통해 사용자가 입력한
• 원문 한문 텍스트
• AI가 생성한 번역 텍스트
• UUID (고유 ID)
등 최소한의 정보를 수집·저장합니다.

2. 개인정보 수집 및 이용 목적

• 사용자가 번역을 요청할 때마다 해당 데이터를 시트에 저장하여
• 이후 유사 예시 검색 및 번역 품질 개선에 활용하기 위함입니다.

3. 개인정보 보유 기간

수집된 데이터는 사용자가 직접 삭제 요청 또는 Google Sheets 상에서 삭제할 때까지 보유됩니다.

4. 개인정보 제3자 제공

본 플러그인은 사용자의 동의 없이 제3자(구글 외 타 기관 포함)에게
개인정보를 제공하지 않습니다.

5. 개인정보 처리 위탁

본 플러그인은 Google Sheets API, OpenAI API 제공사에 번역 데이터 처리를 위탁합니다.
위탁 시에도 OAuth 인증을 통해 사용자가 명시적으로 권한을 부여한 계정의 데이터만 사용됩니다.
